======================================================
                  ADHD-Reader-Project
======================================================

Introduction:
--------------
My Project is a text converter that allows the user to type in text & few of the first letters
of each word get bolded out therefore making it easier for the user to focus when reading.

Usage:
--------------
1. Click the link in Github Pages

2. Paste/Type in the text you want to input

3. Then right click the convert button

4. The converted text with be displayed
